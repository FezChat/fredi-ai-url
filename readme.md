# FEE-XMD WhatsApp Booster

A comprehensive web application for boosting WhatsApp channel followers and group members using the @whiskeysockets/baileys library.

## Features

- **Channel Boosting**: Automatically follow WhatsApp newsletters/channels
- **Group Boosting**: Automatically accept group invites
- **File Upload**: Support for VCF, JSON, and TXT contact files
- **Real-time Progress**: Monitor boosting progress in real-time
- **Group Capacity Check**: Automatically stops if group is full
- **Dashboard Interface**: User-friendly web interface

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- WhatsApp account (for automation)
